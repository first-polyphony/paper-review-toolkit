# Reticle brand mark (public hosting)

Public, durable hosting for the **Reticle wireframe mark** so it can be referenced
from email templates and other online-facing surfaces.

- `reticle_mark_mono_bone_256.png` / `_512.png` — mono **Bone** (`#D8D8F0`) wireframe
  mark on a transparent background, intended for the dark (`#060610`) Reticle email
  header. This is the **Reticle wireframe mark**, not the Polyphony soundwave logo.

Source of truth: `Reticle_brand_assets/png_mono_bone/3q/` in the Reticle Firm assets.

Used by: Supabase auth email templates (project `poubdlllvphcxhtognwr`).

Hosted here (a public repo we control) for a stable, CDN-backed `https` URL with
`image/png` content-type. Do not delete these files without updating the email
templates that reference them.
